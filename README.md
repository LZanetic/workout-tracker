# Workout Tracker

A mobile-first workout tracking application built with React and Spring Boot.

## Features
- Upload workout plans via CSV
- Track sets, reps, and weights
- View progress over time
- Mobile-optimized for gym use

## Tech Stack
- React + Tailwind CSS
- Spring Boot REST API (coming soon)
- PostgreSQL (coming soon)
